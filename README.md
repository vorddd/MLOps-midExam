# Shipping Delay Prediction

A Streamlit-based machine learning application for predicting whether a shipping delivery will arrive on time or late.  
This project is deployed using **HuggingFace Spaces (Docker)** and integrates a trained KNN model with a preprocessing pipeline.

---

## 🚀 Features
- Exploratory Data Analysis (EDA)
- Shipping delay prediction using KNN
- Clean Streamlit interface
- Fully containerized using Docker
- CI/CD through GitHub Actions
